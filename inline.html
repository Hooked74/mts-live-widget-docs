Inline Widget

Встраивание виджета в контент страницы

Главная Inline Modal Link Примеры

Описание

Inline виджет встраивается прямо в контент страницы как iframe. Он остается видимым и доступным для взаимодействия постоянно.

Преимущества:

  • Постоянно видимый виджет
  • Не требует дополнительных действий от пользователя
  • Идеально подходит для страниц событий
  • Полная функциональность покупки билетов

Подключение скрипта

<script> (function () { const rnd = '?' + new Date().getTime() * Math.random(); const script = document.createElement('script'); const target = document.getElementsByTagName('script')[0]; script.async = true; script.src = 'https://stg.live.mts.ru/scripts/widget-v1.iife.js' + rnd; target.parentNode.insertBefore(script, target); })(); </script>

Способы использования

1. JavaScript API

<div id="widget-container"></div> <script> window.addEventListener("load", () => { const widget = new window[Symbol.for("LiveNativeWidgetEmbed")]({ selector: "#widget-container", type: "inline", width: "100%", height: "600px", regionAlias: "moscow", announcementAlias: "mikhail-bublik-g-podolsk", eventId: "26541557", clientId: "your-client-id", }); widget.create(); }); </script>

2. HTML атрибуты

Вариант 1

<live:widget data-client-id="your-client-id" data-type="inline" data-region-alias="moscow" data-announcement-alias="mikhail-bublik-g-podolsk" data-event-id="26541557" data-width="100%" data-height="600px"> </live:widget>

Вариант 2

<script type="text/html" data-selector="live:widget" data-client-id="your-client-id" data-type="inline" data-region-alias="moscow" data-announcement-alias="mikhail-bublik-g-podolsk" data-event-id="26541557" data-width="100%" data-height="600px"> </script>

Параметры

Параметр Тип Обязательный Описание
selector string Да CSS селектор или HTML Element контейнера
type string Да Тип виджета: "inline"
width string Нет Ширина виджета (по умолчанию: 100%)
height string Нет Высота виджета (по умолчанию: 100%)
regionAlias string Да Алиас региона
announcementAlias string Да Алиас анонса
eventId string Да ID события
clientId string Да ID клиента
selectFrameByInstance boolean Нет Уникальный iframe для каждого виджета
