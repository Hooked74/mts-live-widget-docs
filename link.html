Link Widget

Открытие виджета в новой вкладке

Главная Inline Modal Link Примеры

Описание

Link виджет открывает страницу покупки билетов в новой вкладке браузера. Самый простой способ интеграции.

Преимущества:

  • Простейшая интеграция
  • Не влияет на производительность основной страницы
  • Полная функциональность в отдельной вкладке
  • Нет ограничений по размеру

Подключение скрипта

<script>
  (function () {
    const rnd = '?' + new Date().getTime() * Math.random();
    const script = document.createElement('script');
    const target = document.getElementsByTagName('script')[0];
    script.async = true;
    script.src = 'https://stg.live.mts.ru/scripts/widget-v1.iife.js' + rnd;
    target.parentNode.insertBefore(script, target);
  })();
</script>

Способы использования

1. JavaScript API

<button id="buy-tickets">Купить билеты</button>

<script>
  window.addEventListener("load", () => {
    const widget = new window[Symbol.for("LiveNativeWidgetEmbed")]({
      selector: "#buy-tickets",
      type: "link",
      regionAlias: "moscow",
      announcementAlias: "drug-na-chas-komediya",
      eventId: "27105847",
      clientId: "your-client-id",
    });
    widget.create();
  });
</script>

2. HTML атрибуты

Вариант 1

<live:widget 
  data-client-id="your-client-id" 
  data-type="link" 
  data-region-alias="moscow"
  data-announcement-alias="drug-na-chas-komediya" 
  data-event-id="27105847" 
  data-tag-name="button"
  class="btn">
  Купить билеты
</live:widget>

Вариант 2

<script 
  type="text/html"
  data-selector="live:widget"
  data-client-id="your-client-id" 
  data-type="link" 
  data-region-alias="moscow"
  data-announcement-alias="drug-na-chas-komediya" 
  data-event-id="27105847" 
  data-tag-name="button"
  class="btn">
  Купить билеты
</script>

Параметры

Параметр Тип Обязательный Описание
selector string Да CSS селектор кнопки/элемента
type string Да Тип виджета: "link"
regionAlias string Да Алиас региона
announcementAlias string Да Алиас анонса
eventId string Да ID события
clientId string Да ID клиента
