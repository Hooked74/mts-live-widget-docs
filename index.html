Быстрая интеграция виджета

Добавьте виджет покупки билетов на ваш сайт всего за несколько минут. Поддерживаются различные типы интеграции.

Подключение скрипта

Добавьте следующий код в ваш HTML:

<script> (function () { const rnd = '?' + new Date().getTime() * Math.random(); const script = document.createElement('script'); const target = document.getElementsByTagName('script')[0]; script.async = true; script.src = 'https://stg.live.mts.ru/scripts/widget-v1.iife.js' + rnd; target.parentNode.insertBefore(script, target); })(); </script>

Inline Widget

Встраивает виджет прямо в контент страницы как iframe. Идеально подходит для постоянного отображения.

Modal Widget

Открывает виджет в модальном окне по клику на кнопку. Не занимает место на странице.
